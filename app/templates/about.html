_(┐「ε:)_   The Tools We Used   _(┐「ε:)_

  • Python

    • Python is one of the languages used to help bring our application to life. Its functionality allows for easy data manipulation. In combination with other tools, its syntax provides for easier interpretation and cleaner presentation.

  • Flask

    • Flask is a microframework for Python and serves as the foundation for our application. It uses Jinja2 templating and RESTful request dispatching. This tool helped facilitate our web application development process overall.

  • Bootstrap

    • Bootstrap is an open source framework that uses HTML, CSS, and JS. It helped both speed up and ease our front end development because of how flexible it is.

  • Piccolo

    • Piccolo is a bootstrap HTML template. We implemented this tool to enhance the presentation of our application and give it a more aesthethically pleasing appearance.

  • nginx

    • nginx is a high-performance HTTP server and reverse proxy. This tool was used to serve our application by handling the requests sent by our API.

  • uwsgi

    • uWSGI is a deployment option for severs. Because we decided to use nginx, combining it with uWSGI gives us more flexibility and better performance because of the configurations it offers.

  • Apiary

    • Apiary is a tool that allowed for an easier, faster and more efficient creation of our API. It offers different tools to prototype your API and configure it to your liking.

  • Github

    • Github is a web-based Git repository. Since the project is being worked on by several team members concurrently, it allows for more efficient contributions towards the overall project. In addition, it helps simulate a real world environment of software development and version control.

  • Rackspace

    • Rackspace has allowed our team to host our application using one of their virtual machines which has 1gb RAM and 20gb of storage.
